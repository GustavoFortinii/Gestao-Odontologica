# Gestão-Odontologica

Projeto referente à matéria Fundamentos de Sistemas de Informação: Aplicações Web. 
O tema escolhido foi "Gestão Odontologica". Que visa solucionar problemas na area de Odontologia, como agendamento de consultas, cadastro de informaçoes dos clientes, disponibilizar informaçoes educativas e entre outros fatores.
