# Gestao-Odontologica

Projeto referente à matéria Fundamentos de Sistemas de Informação : Aplicações Web. O tema escolhido foi "Gestão Odontologica", que visa solucionar problemas na area de Odontologia, agendamento da consulta, cadastro das informaçoes do cliente, disponibilizar informaçoes educativas de melhor cuidado e entre outros fatores.
